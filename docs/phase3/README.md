## 快速开始

```bash
# 1. 安装依赖
cd quanys-blog
npm install

# 2. 安装 Element Plus
npm install element-plus @element-plus/icons-vue

# 3. 安装 Markdown 相关库
npm install marked highlight.js @kangc/v-md-editor

# 4. 启动开发服务器
npm run dev
```
