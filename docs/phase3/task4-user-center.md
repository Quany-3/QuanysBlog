# Task 4: 用户中心

## 目标
开发用户个人资料页面，提供个人信息管理功能。

---

## 4.1 个人资料页 Profile.vue

### 方案
表单展示和编辑用户基本信息。

### 执行流程
1. 加载当前用户信息
2. 实现表单展示（用户名、邮箱、头像、个人简介）
3. 实现头像上传
4. 实现个人信息更新

### 拆分
| 区块 | 内容 |
|------|------|
| 基本信息 | 用户名、邮箱、个人简介 |
| 头像 | 头像上传和预览 |
| 操作 | 保存修改 |

### 验证标准
- 页面显示当前用户信息
- 可修改个人简介
- 可上传新头像
- 保存后数据更新
